# Code smells and how to address them
The purpose of this project is to tackle common code smells and ways to refactor them.

## Code smells covered:
* Poor Names
* Poor Naming Conventions
* Poor Method Signatures
* Long Parameter List
* Output parameters
* Variable Declaration on top
* Magic numbers
* Nested Contidionals
* Switch Statements
* Duplicated Code
* Comments
* Long Methods
